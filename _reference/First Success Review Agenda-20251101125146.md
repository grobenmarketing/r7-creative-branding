# First Success Review Agenda

**Duration:** 15 minutes
**Meeting Type:** Early Value Activation
**When:** Within 48 hours of ClickUp Client Folder Created
* * *
## Purpose
Get client comfortable with their workspace and establish early confidence in the system.
* * *
## Meeting Agenda
### 1\. Welcome to Your Workspace (3 min)
*   Share screen showing their ClickUp folder structure
*   Show where Brand Pillars will appear
*   Show Content list (where assignments will live)
### 2\. Walk Through One Sample Task (5 min)
**Create a dummy task together:**
*   "Let's create a practice content assignment so you see how this works"
*   Add: Title, Owner, Due Date, Description
*   Show how to change status
*   Show calendar view of tasks
### 3\. Access Confirmation (3 min)
**Verify they can access:**
*   ClickUp (have them click into their folder while on call)
*   [Frame.io](http://Frame.io) shared folder (send test link, have them confirm they can see it)
*   JotForm link (they should bookmark this)
### 4\. Set Expectations & First Value Milestone (3 min)
*   Brand Strategy Meeting coming up
*   **First Value Delivery:** Brand Pillars document (2 days after Brand Strategy)
    *   "This is a professional copywriting asset you can use everywhere - website, sales materials, marketing. It clarifies your positioning and messaging."
*   **Second Value Delivery:** First content published (3-4 weeks out)
*   Content Strategy will be where we assign real work
### 5\. Answer Questions (1 min)
* * *

## After This Call
### Update ClickUp:
In Account Management > Implementation Checklist:
*   Check off: ✅ "Quick Win Call completed"
*   Add note: "Client confirmed access to all systems"
### Flag Issues:
If client couldn't access:
*   ClickUp → Resend invitation, verify email address
*   [Frame.io](http://Frame.io) → Check shared link permissions
*   JotForm → Verify link is working
* * *

## Success Checklist
✅ Client successfully navigated to their ClickUp folder
✅ Client can see and edit a task
✅ [Frame.io](http://Frame.io) access confirmed
✅ JotForm bookmarked
✅ No technical blockers identified
